# Installation

# Clone Repository

```bash
git clone https://github.com/your-username/advanced-ai-demand-forecasting.git
```

# Navigate

```bash
cd AI-demand-forecasting-phase_4
```

# Install Dependencies

```bash
npm install
```

# Start Development Server

```bash
npm run dev
```

